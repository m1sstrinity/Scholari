## üöÄ Key Features
* **Secure Authentication:** Protects user data through an OTP (One-Time Password) or password-based login system.

* **Task Management Dashboard:** Allows educators to distribute tasks with multimedia elements (PDFs/Images) and students to monitor deadlines.

* **Progress Tracker:** An analytics dashboard that displays finished/unfinished work and performance indicators.

* **Real-Time Collaboration:** In-app notifications and status updates to keep students and teachers synchronized.

* **Role-Based Access:** Customized permissions and views tailored specifically for either Student or Educator roles.
